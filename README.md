# Fundão Tech Solutions

Projeto desenvolvido com foco em prática de desenvolvimento web utilizando HTML e CSS.

## Sobre o Projeto

A Fundão Tech Solutions é uma landing page criada para apresentar uma equipe fictícia de tecnologia, seus serviços e informações de contato.

O objetivo principal do projeto é praticar:

- Estruturação HTML semântica
- Estilização com CSS
- Organização de layout
- Criação de seções responsivas
- Componentização visual
- Estrutura de páginas web modernas

---

## Tecnologias Utilizadas

- HTML5
- CSS3

---

## Funcionalidades

- Hero Section moderna
- Cards de equipe
- Seção de serviços
- Área institucional
- Contato
- Layout organizado e responsivo
